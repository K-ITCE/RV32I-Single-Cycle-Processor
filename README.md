# RV32I-Single-Cycle-Processor

Disclaimer: I'm currently in the process of organizing the project's files and documentation before uploading. Last updated: April 4th, 2026, 2:30AM.

My partial implementation of a 32-bit single cycle RISC-V processor built using VHDL in Intel Quartus Prime. So far, it successfully performs 22 different instructions including data movement, arithmetic, and branch.
